# v5_random Command Logs

This folder stores command logs for the `v5_random` Roundtrip alignment run.

- Primary command wrapper: `bash scripts/run_v5_eval_metric.sh roundtrip_alignment v5_random 0`.
- Log use: inspect only for execution provenance and failures.
- Reporting rule: final numeric values must come from parsed outputs in `3_evaluation/09_aggregation/`, not from live logs.
