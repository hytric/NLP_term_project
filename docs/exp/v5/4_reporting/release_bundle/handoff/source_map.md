Rule for final edits:

Only replace table values from measured artifacts. If a metric is blocked,
write `blocked` or `coverage-limited` rather than leaving a silent blank.
live logs and partial files cannot become final result claims.
target10 downstream claims remain coverage-limited while retained downstream
target10 task coverage is `0/10`; do not promote them from available-language rows.
