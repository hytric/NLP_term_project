Generated by `scripts/build_v52_similarity_pairs.py`.

Total pairs: `2100`

## Purpose

This analysis checks whether sentence vectors become organized by meaning, language,
and roundtrip alignment structure after continued MLM training and embedding
initialization. It is a novelty/diagnostic analysis, not an official Glot500 task.

Main questions:

- Same meaning: are target-language sentences close to their English aligned sentence?
- Same language: do target-language sentences cluster together even when the meaning differs?
- Roundtrip: how close are source-English, source-pivot, and English-pivot vectors?
- 2D map: do models separate by language identity or align by semantic content?

Interpretation boundary:

- High aligned-pair similarity supports semantic alignment.
- High same-language-adjacent similarity supports language/style clustering, not necessarily semantic alignment.
- Roundtrip pairs should be interpreted as multi-way alignment diagnostics.
- The PCA map is qualitative; use the TSV summaries for numeric comparison.
